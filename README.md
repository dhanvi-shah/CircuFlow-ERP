# CircuFlow-ERP
CircuFlow ERP is a circular economy platform that streamlines waste collection, recycling, and resale. It connects users, agencies, sellers, and buyers through an integrated system, transforming waste into valuable resources via intelligent tracking, inventory management, and a sustainable marketplace.

## 🚀 Overview

Every year, billions of tons of waste are generated, yet only a small fraction is efficiently recycled. CircuFlow ERP bridges this gap by connecting users, recycling agencies, sellers, and buyers into one seamless ecosystem.

Our platform transforms waste into valuable resources through structured workflows, real-time tracking, and a sustainable marketplace.

---

## 🌍 Key Features

### 🔹 Recycling Management
- Schedule waste pickups
- Track request status (Pending → Assigned → Completed)
- Categorize waste by type and quantity

### 🔹 Warehouse & Inventory
- Centralized storage system
- Stock tracking based on waste type
- Kanban-style classification for better visualization

### 🔹 Marketplace System
- Sellers list recycled products
- Buyers browse and purchase items
- Automatic stock updates after purchase
- Out-of-stock products become inactive

### 🔹 Modular ERP Design
- Built within a single Odoo app
- Separate models for Buyers and Sellers
- Scalable and customizable architecture

## 🛠️ Tech Stack

- Odoo Framework

## 💡 How It Works

1. Users request waste pickup  
2. Agencies collect and store waste in warehouses  
3. Waste is categorized and tracked as inventory  
4. Sellers create products from recycled materials  
5. Buyers purchase products through the platform
